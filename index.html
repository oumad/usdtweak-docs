Welcome to usdtweak

Ultra lightweight
USD editor

usdtweak is a free and open source editor for Pixar's USD format.

Download (alpha) Read the docs

Blazing fast startup and file loading times

usdtweak is written in C++ and relies entirely on GPU for the UI, no more bloated python packages overhead, allowing for the fastest startup and a snappy UI.

Unmediated USD based editing

usdtweak provides a simple UI for a direct 1:1 corresspondance to native USD features, no runtime translations or obscure abstractions, ensuring the most interoperable and durable workflow.

Multi Layer and stage editing

usdtweak can efficiently load multiple layers and stages in memory at a time, allowing the user to quickly switch between them when editing layers individually or authoring them directly from a stage.
