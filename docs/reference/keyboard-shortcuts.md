# Keyboard Shortcuts

A reference of all keyboard shortcuts available in usdtweak.

## General

| Shortcut | Action |
|----------|--------|
| **Ctrl+S** | Save the current layer |
| **Ctrl+F** | Save the current layer as (new file) |
| **Ctrl+Z** | Undo |
| **Ctrl+R** | Redo |
| **Ctrl+X** | Cut (when available) |
| **Ctrl+C** | Copy (when available) |
| **Ctrl+V** | Paste (when available) |

## Viewport navigation

| Shortcut | Action |
|----------|--------|
| **Middle mouse button + drag** | Orbit the camera |
| **Shift + Middle mouse button + drag** | Pan the camera |
| **Scroll wheel** | Zoom in/out |

### Fly mode

| Shortcut | Action |
|----------|--------|
| **W** | Move forward |
| **S** | Move backward |
| **A** | Strafe left |
| **D** | Strafe right |

## Viewport manipulators

| Shortcut | Action |
|----------|--------|
| **Left-click** (in viewport) | Select a prim |
| **Shift + scale drag** | Uniform scale |

## Application

| Shortcut | Action |
|----------|--------|
| **File > Quit** | Close the application (prompts to save unsaved work) |

:::info
Keyboard shortcuts are currently defined at compile time and cannot be customized through the UI. This may change in future versions.
:::
